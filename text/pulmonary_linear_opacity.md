# Pulmonary linear opacity—`OIFM_OIDM_062845`

**Synonyms:** linear density, band-like opacity, linear shadow, linear marking, bandlike opacity, linear opacities, linear opacity

**Tags:** chest, XR, finding

A thin, elongated opacity on chest radiograph that may represent atelectasis, scarring, a Kerley line, or other linear pathology.
