# Flow in the ascending aorta—`OIFM_CDE_000031`

This module describes the common data elements for flow in the ascending aorta use case

**Codes:** RADELEMENT RDES31 flow in the ascending aorta; RADLEX RID480 aorta

## Attributes

### Flow rate—`OIFMA_CDE_000209`

Measurement of net flow volume in the ascending thoracic aorta above the aortic valve mL/sec  
Mininum: 0  
Maximum: 100  
Unit: ml per sec

### Gradient—`OIFMA_CDE_000210`

Pressure of aortic flow Pa

Maximum: 100  
Unit: Pa
